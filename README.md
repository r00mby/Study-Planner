# Study Planner

A web-based Study Planner developed using PHP, HTML, CSS, and MySQL.

## Features
- Add new tasks
- Edit tasks
- Delete tasks
- View all tasks

## Technologies Used
- PHP
- HTML
- CSS
- MySQL

## Database
The SQL database file is located in the Database folder.

## How to Run
1. Install XAMPP.
2. Copy the project to the htdocs folder.
3. Import study_track_db.sql into phpMyAdmin.
4. Start Apache and MySQL.
5. Open the project in your browser.
